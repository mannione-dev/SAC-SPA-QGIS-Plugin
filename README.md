# SAC-SPA-QGIS-Plugin
A QGIS plugin to list species and habitats for SAC/SPA sites by clicking on a QGIS layer.


How to make the plugin zip to install into qgis.

Step 1: Zip the folder
Do not zip the files individually. Zip the parent folder itself.

Option A: Using the Command Line (Bash)
If you want to do this quickly in the terminal, navigate to the directory containing your parent folder and run:

Bash
zip -r sac_spa_species_habitats.zip sac_spa_species_habitats

Option B: Using Windows
Right-click the sac_spa_species_habitats folder.

Select Send to > Compressed (zipped) folder.

Option C: Using Mac
Right-click (or Ctrl+click) the sac_spa_species_habitats folder.

Select Compress "sac_spa_species_habitats".
